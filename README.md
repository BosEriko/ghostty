# Ghostty (Terminal)
Download [Brew](https://github.com/BosEriko/brew) as your package manager then install [Ghostty](https://formulae.brew.sh/cask/ghostty).

## Install Ghostty
```sh
brew install --cask ghostty
```

## Install the config
Make sure to remove or move your current `Ghostty` directory
```sh
git clone https://github.com/BosEriko/ghostty.git ~/.config/ghostty
```
